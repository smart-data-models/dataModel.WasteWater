<!-- 20-Description -->    
Descripción global: **Esta entidad contiene una descripción armonizada de un Tanque genérico realizado para el ámbito del tratamiento de Aguas Residuales. Para un determinado tipo de tanque, se enumeran como propiedades todas las posibles variables que pueden ser medidas. En la propiedad de descripción puede definirse el tipo de depósito (anaerobio, predinitrificación, nitrificación, etc.)**.    
versión: 0.1.0
